## Prerequisites

- Docker
- Docker Compose

## Running the project

1. Clone the repository
2. Run `docker-compose up --build`

## Open the project

1. Frontend is available at `http://localhost:3000`
2. Backend API is available at `http://localhost:5001/v1/swatches`
